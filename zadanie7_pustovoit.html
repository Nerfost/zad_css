1. Na co pozwala styl lokalny, który element będzie formatowany z użyciem stylu lokalnego
  2. Składnia teoretyczna ( z użyciem ramki otaczającej→ może być tabela o jednej komórce, zastosuj „znaki specjalne” do napisania definicji stylu lokalnego tak aby instrukcja CSS nie wykonywała się a tylko wyświetlała wewnątrz ramki [  spacja niełamana] przykład „znaku specjalnego”)
  3. Co może być selektorem
  4. Co wipisujemy jako "cechę". Jakie jest znaczenie średników
  5. Co wipisujemy jako "wartość" (w deklaracji stylu)

  1. Styl lokalny pozwala na nadanie formatowania konkretnemu pojedynczemu elementowi strony.
  2. Składnia teoretyczna ( z użyciem ramki otaczającej→ może być tabela o jednej komórce, zastosuj „znaki specjalne” do napisania definicji stylu lokalnego tak aby instrukcja CSS nie wykonywała się a tylko wyświetlała wewnątrz ramki [  spacja niełamana] przykład „znaku specjalnego”)
  3. Istnieje wiele różnych rodzajów selektorów, takich jak selektory elementów, klas, identyfikatorów, atrybutów i wiele innych.
  4. Jako "cecha" czy inaczej własność bądź właściwość (ang. "property") należy wpisać o jakie konkretnie atrybuty formatowania nam chodzi (np. kolor tekstu - color). Jednemu selektorowi możemy nadać kilka atrybutów (cech). Są one wtedy rozdzielone średnikami.
  5. Jako wyraz "wartość" (w deklaracji stylu) wpisujemy dokładną wartość atrybutu (np. dla koloru tekstu będzie to: red, blue czy też #31F4A5 itd.). Wartość zależy ściśle od cechy, po której stoi w deklaracji.
